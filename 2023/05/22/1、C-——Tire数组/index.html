现在就来看一下我对于a[[p]][][u]的理解：

对于insert（）函数，root位置就是0，而图中所标的数字便是通过insert（）操作给a[[p]][][u]赋的数（这个很好模拟出来），而在query（）查询操作时就是从上往下通过p来查找的过程。

比如，就以插入adef操作时(此时idex = 6)：

首先，第一个位置a之前存在，so直接p = a[[1]][][0],接着看d的a[[1]][][3]这个位置是不是0（即存不存在），因为不存在，so呀a[[1]][][3] = 7，再将p = 7；

接着，就是看a[[7]][][4] = 8,p = 8;

然后a[[8]][][5] = 9,p = 9;

而最后的一步cnt[9] ++,就是表示这种字符串出现次数+1咯。
